## Ollama

- ollama run <Modell> => führt ein Modell aus zum prompten
- ollama pull <Modell> => lädt ein Modell herunter
- ollama ls => listet alle lokalen Modelle auf
- ollama ps => zeigt die aktuell aktiven Modelle an

## Modelle

- llama3.2 - Facebook/Meta
- gemma4 - Google
- qwen - Alibaba
- granite
